# Intalação
## Apenas pip
```shell
python3 -m pip install -r requirements.txt
```

## Usando virtual env
```shell
python -m venv myenv
source myenv/bin/activate
pip install -r requirements.txt
```

## Usando miniconda
```shell
conda create -n webScrap --file requirements.txt
conda activate webScrap
```
